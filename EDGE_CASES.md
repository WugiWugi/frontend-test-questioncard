# QuestionCard Edge Cases & UX

- Explanation отсутствует → показать "Explanation отсутствует"
- В stem только формулы → рендер через KaTeX без текста
- Очень длинный текст → скролл или обрезка
- KaTeX упал с ошибкой → fallback текст "Ошибка формулы"
- Пользователь меняет ответ после check → выбор заблокирован
- Demo режим → скрыть/blurr Explanation, показать "Доступно только в полной версии", CTA "Оплатить"
